# cs2-autoteam
*simple team assignment plugin for counterstrikesharp*

## Automatically places to the team specified via config
```json
{
  "Prefix": "{red}[AutoTeam]",
  "TeamToBeAssigned": 2, // Terrorist = 2 | Counter Terrorist = 3 | Spectator = 1
  "DisableJoinFromTeamMenu": true,
  "ConfigVersion": 1
}
```
